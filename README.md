# TCS Stock Analysis

## Project Overview
This project analyzes historical TCS stock data using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- SQL

## Analysis
- Data Cleaning
- Year-wise Average Close Price
- Trading Volume Analysis
- Daily Returns
- Moving Averages (50-Day & 200-Day)

## Key Insights
- A large company like tcs has a stable stock flow over the years with certain localised significant changes
- The Crash of 2008-09 saw the stock value dip but it picked up linearly
- Major Highs were obtained after corona crash in 2020 last quarter and beginning 2021
- TCS stock price picked up almost vertically after 2020 21
